1. Overview

An advanced Odoo 18 module enhancing product search by:

  • Combining product names in all installed languages
  • Multi-word partial search
  • Full-text ranking for relevance
  • Ultra-fast cache table
  • Automatic trigger & cron updates
  • Optimized for website & backend search

2. Installation

  1. Copy module ZIP to Odoo addons directory
